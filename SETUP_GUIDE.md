This is a complete Playwright Test Runner Platform with React frontend and Express backend.

## Prerequisites
- Node.js >= 18.x
- PostgreSQL >= 14.x
- npm >= 9.x

## Backend Setup

1. Navigate to backend:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
npx playwright install
```

3. Configure `.env` file with your database credentials

4. Start backend:
```bash
npm run dev
```

Backend runs on http://localhost:5000

## Frontend Setup

1. Navigate to frontend:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start frontend:
```bash
npm run dev
```

Frontend runs on http://localhost:5173

## Quick Usage

1. Register/Login at http://localhost:5173
2. Create a project
3. Create test suites
4. Write test cases with Playwright code
5. Run tests and view HTML reports
